It is a movie recommendation model. 

The data set used was MovieLens 10m, that was divided into edx and validation. 

Edx and validation files are available at folder "data". (they could not be uploaded bc they are bigger then 25MB

Other files made through the code are available at folder "rdas". (they could not be uploaded bc they are bigger then 25MB

At report.Rmd are plots codes and links to its images saved at folder "figs". It generates the final report in pdf

At code.R you will find the algorithm that makes the predictions, saves the partial files and prints the final RMSE.
